## TABLE
+ DB는 2차원의 TABLE형태로 data를 저장한다.
+ 크기는 Byte 단위이다.


|이름|용어|
|---|---|
|테이블|**ENTITY**|
|가로|**ROW / INSTANCE**|
|세로|**COLUMN / 속성(ATTRIBUTE)**|
|첫번째 COLUMN|**COLUMN행 또는 COLUMN HEADER**|
|Data 한개|**COLUMN값, 필드(FIELD)**|
