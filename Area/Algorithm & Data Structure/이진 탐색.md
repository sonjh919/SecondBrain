+ index같은 내장함수의 경우 순차검색(O(n))이기 때문에 코테에서 시간 초과가 나는 경우가 많다. 
+ O(logN)의 시간 복잡도를 보장하는 이진 탐색 코드를 외워두고 써먹자.

```python
def binary_search(arr, start, target, end):  
	while start<=end:  
		mid = (start+end)//2  
		if arr[mid] == target:  
			return mid  
		elif arr[mid] > target:  
			end = mid-1  
		else:  
			start = mid+1  
	return None
```
