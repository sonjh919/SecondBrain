## Network
+ 네트워크는 여러대의 컴퓨터 또는 장비가 서로 연결되어서 정보를 주고 받을 수 있게 도와주는 기술이다.
- **컴퓨터, 라우터, 스위치, 허브** 등의 장비들이 각각의 역할을 수행하여 정보를 주고 받는다.
- 서로 정보를 주고 받기 위해서는 IP 주소, 서브넷 마스크, 게이트웨이등의 정보를 설정하고 **네트워크 프로토콜을 이용하여 통신**을 하게 된다.
