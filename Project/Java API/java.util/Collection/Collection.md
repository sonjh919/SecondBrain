## 🌈 Java.util.Collection
+ 자료구조와 알고리즘들이 미리 구현되어 있는 패키지
+ [[배열]]이 가진 고정 크기의 단점을 극복하기 위해 객체들을 쉽게 삽입, 삭제, 검색할 수 있는 **가변 크기**의 컨테이너이다.
+ 컬렉션의 요소로는 **객체만 가능**하지만, 기본 타입의 값이 들어오면 auto boxing에 의해 [[Wrapper]] 클래스로 변환된다.
+ [[제네릭]] 기법으로 구현되어 있다.
