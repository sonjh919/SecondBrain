## Error
+ **프로그램이 수행 중에 발생하는 심각한 문제**로, **시스템 수준의 오류**를 나타낸다.
+ 이러한 문제는 일반적으로 **프로그램이 복구할 수 없는 상황**을 가리킨다.
+ 개발자가 Error를 처리하는 것은 권장되지 않으며, 시스템 레벨에서 처리되어야 한다.
