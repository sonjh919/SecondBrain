## 🌈 UNION ALL
- A ∪ B
- 중복된 행도 그대로 출력
- **별칭은 첫번째 TABLE을 기준으로 표시**
- **1대1 관계에서 결과는 2배가 됨**

```sql
SELECT * FROM RUNNING_MAN
UNION ALL
SELECT * FROM INFINITE_CHALLENGE;
```

## 🌈 UNION
- A ∪ B
- 중복된 행은 제거

```sql
SELECT * FROM RUNNING_MAN
UNION
SELECT * FROM INFINITE_CHALLENGE;
```

## 🌈 INTERSECT
- A ∩ B

```sql
SELECT * FROM RUNNING_MAN
INTERSECT
SELECT * FROM INFINITE_CHALLENGE;
```

## 🌈 MINUS/EXCEPT
- A - B

```sql
SELECT * FROM RUNNING_MAN
MINUS
SELECT * FROM INFINITE_CHALLENGE;
```
