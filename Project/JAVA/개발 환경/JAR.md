## JAR
+ 프로그램의 빌드된 실행 가능한 코드를 포함하는 패키지이다.
+ Java 애플리케이션의 클래스, 리소스, 라이브러리 등을 포함하며, 실행 가능한 Java 애플리케이션을 실행하기 위해 사용된다.
+ 디렉토리 구조나 빌드 시스템에 따라 위치가 다를 수 있다.
+ Maven은 target, gradle은 build, Intellij는 out에 위치한다.
