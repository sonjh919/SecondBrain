## Class
+ Java에서 클래스와 인터페이스에 대한 메타데이터 정보를 나타낸다.
+ Reflection(리플렉션)을 지원한다.
+ 실행 중인 Java 애플리케이션에서 클래스에 대한 정보를 동적으로 검사하고 조작할 수 있게 해준다.

### Class 주요 기능
1. 클래스의 메타데이터 제공
2. 동적으로 클래스 로딩 및 인스턴스 생성
3. 리플렉션을 통한 클래스 정보 검사 및 조작
4. 제네릭 타입 정보 제공
