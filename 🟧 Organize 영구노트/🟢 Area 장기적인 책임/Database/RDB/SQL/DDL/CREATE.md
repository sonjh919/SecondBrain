## CREATE

- DATETIME은 별도의 크기지정이 없다.
- [[제약조건]]을 추가할 수 있다.

```sql
CREATE TABLE 'TABLE명' (
	'칼럼명1' DATATYPE 'DEFAULT형식',
	'칼럼명2' DATATYPE 'DEFAULT형식'
CONSTRAINT '제약조건명' '제약조건' ('컬럼명')
);
```

## 사용자 생성
+ 사용자 생성 권한이 있어야 가능하다.

CREATE USER '사용자명' IDENTIFIED BY '패스워드';
