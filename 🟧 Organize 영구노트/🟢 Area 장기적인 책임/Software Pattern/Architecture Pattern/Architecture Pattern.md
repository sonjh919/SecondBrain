## Architecture Pattern
- Architecture Pattern은 시스템의 전체 구조와 구성 요소 간의 상호작용에 초점을 둔다.
- **시스템의 전체적인 설계와 구조**에 대한 고수준의 패턴이다.
- 시스템의 확장성, 성능, 보안, 유지보수성 등과 같은 **비기능적 요구사항**을 충족시키기 위해 사용된다.
