#Java 

## 자바의 대격변, Java 8
### 자바가 직면했던 새로운 요구사항들
+ 자바는 JVM 덕분에 높은 이식성과 객체지향으로 사랑받았으나, 시간이 지나며 새로운 요구사항들이 등장하게 된다.
+ 빅데이터를 처리할 필요성이 늘어났고, 멀티코어 컴퓨터와 같은 병렬 프로세싱이 가능한 장비들이 보급되면서 **병렬 처리**의 필요성이 늘어났다.
+ [[함수형 프로그래밍]]은, 객체 지향 프로그래밍처럼 프로그래밍의 패러다임의 한 종류이다.
