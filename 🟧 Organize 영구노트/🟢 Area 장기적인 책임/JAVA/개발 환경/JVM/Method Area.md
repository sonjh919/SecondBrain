## Method Area
+ = **Metaspace, Static Area, Class Area**
- 클래스 정보를 처음 메모리 공간에 올릴 때 초기화 되는 대상을 저장하기 위한 공간이다.
- 클래스의 타입, 변수 및 메서드등의 정보를 가지고 있으며 **모든 [[Thread]]가 공유**한다.
