#Java 


## 메서드 (Method)
- 특정 작업을 수행하기 위한 명령문의 집합

## 메서드 선언
- 메서드명은 동사형으로 작성하여 목적을 쉽게 알수 있게 하는 것이 좋다.

```java
[접근 제한자] [Return type] [함수명]{
       return [return값];
}

// ex)
public void method1(){
	System.out.println("method1이 실행됩니다.");
}
```
