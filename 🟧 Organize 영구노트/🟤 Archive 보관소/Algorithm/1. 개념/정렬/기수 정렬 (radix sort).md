#정렬 #Okn

+ 값을 비교하지 않는 특이한 정렬
+ 값을 놓고 비교할 자릿수를 정한 다음 해당 자릿수만 비교
+ **시간 복잡도가 가장 짧은 정렬**
+ 정렬해야 하는 데이터의 개수가 너무 많을 때 사용

### 기수 정렬 핵심 이론
+ 10개의 큐를 이용한다. 각 큐는 값의 자릿수를 대표한다.
1. 일의 자릿수 기준으로 배열 원소를 큐에 집어넣는다.
2. 0번째 큐부터 9번째 큐까지 pop을 진행한다.
3. 십의 자릿수를 기준으로 같은 과정을 진행한다.
4. 마지막 자릿수를 기준으로 정렬할 때까지 앞의 과정을 반복한다.
