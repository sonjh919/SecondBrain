
### 1. 자료구조 선택.
`키 큰 사람이 -> 몇명인지` 
연관되는 두 데이터가 있다. 이를 받는 방식은 map 또는 배열이 있을 것 같은데, 키처럼 연속적인 데이터가 있기 때문에 [[인덱스가 연속적]]인 배열을 쓰는 것이 맞다!


```java
arr[키] = 나보다 키 큰 사람이 왼쪽에 몇명 있었는지
```


```java
arr[1] = 2
arr[2] = 1
arr[3] = 1
arr[4] = 0
```

### 2. 출력
첫째 줄에 줄을 선 순서대로 키를 출력한다.
arr의 값으로 오름차순하기!
-> 1이 겹치는데...
-> [[배열은 고정 크기]]이기 때문에 힘들다!!
-> 중간에 값을 넣어서 인덱스가 달라지는 리스트를 쓰자! -> 먼저 넣으면 그 이후는 뒤로 밀림.
```java
List<Integer> list = new ArrayList<>();  
for (int i = arr.length - 1; i > 0; i--) {  
    list.add(arr[i], i);  
}
```
