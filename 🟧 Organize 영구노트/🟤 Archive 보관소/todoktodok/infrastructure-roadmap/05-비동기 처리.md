# 비동기 처리

## 1만 명 단계
### 필요성 낮음
- 동기 처리로 충분
- 간단한 작업만 있다면 @Async로 충분

```java
@Configuration
@EnableAsync
public class AsyncConfig {
    @Bean
    public Executor taskExecutor() {
        ThreadPoolTaskExecutor executor = new ThreadPoolTaskExecutor();
        executor.setCorePoolSize(5);
        executor.setMaxPoolSize(10);
        executor.setQueueCapacity(100);
        executor.setThreadNamePrefix("async-");
        executor.initialize();
        return executor;
    }
}

@Service
public class NotificationService {
    @Async
    public CompletableFuture<Void> sendNotification(Long userId, String message) {
        // 알림 발송 로직
        return CompletableFuture.completedFuture(null);
    }
}
```

## 10만 명 단계
### 메시지 큐 도입 검토
**RabbitMQ 또는 Kafka**

#### RabbitMQ 사용 예시
```java
@Configuration
public class RabbitMQConfig {
    @Bean
    public Queue notificationQueue() {
        return new Queue("notification.queue", true);
    }
    
    @Bean
    public TopicExchange exchange() {
        return new TopicExchange("notification.exchange");
    }
}

// Producer
@Service
public class RoomService {
    @Autowired
    private RabbitTemplate rabbitTemplate;
    
    public Room createRoom(RoomDto dto) {
        Room room = roomRepository.save(dto.toEntity());
        
        // 비동기 알림
        NotificationMessage msg = new NotificationMessage(room);
        rabbitTemplate.convertAndSend("notification.exchange", 
                                       "room.created", 
                                       msg);
        return room;
    }
}

// Consumer
@Component
public class NotificationConsumer {
    @RabbitListener(queues = "notification.queue")
    public void handleNotification(NotificationMessage message) {
        // 알림 발송 처리
    }
}
```

## 1000만 명 단계
### Kafka 기반 이벤트 드리븐 아키텍처

#### Kafka 토픽 설계
```
room-events          (토론방 생성/수정/삭제)
comment-events       (댓글 이벤트)
notification-events  (알림 이벤트)
analytics-events     (분석 이벤트)
```

#### Producer 구현
```java
@Service
public class RoomEventProducer {
    @Autowired
    private KafkaTemplate<String, RoomEvent> kafkaTemplate;
    
    public void publishRoomCreated(Room room) {
        RoomEvent event = RoomEvent.builder()
            .eventType("ROOM_CREATED")
            .roomId(room.getId())
            .timestamp(Instant.now())
            .build();
            
        kafkaTemplate.send("room-events", 
                          room.getId().toString(), 
                          event);
    }
}
```

#### Consumer 구현 (Multiple Services)
```java
// Notification Service
@KafkaListener(topics = "room-events", groupId = "notification-group")
public void handleRoomEvent(RoomEvent event) {
    if (event.getEventType().equals("ROOM_CREATED")) {
        sendNotificationToSubscribers(event);
    }
}

// Analytics Service
@KafkaListener(topics = "room-events", groupId = "analytics-group")
public void handleRoomEvent(RoomEvent event) {
    updateStatistics(event);
}

// Search Service (Elasticsearch 동기화)
@KafkaListener(topics = "room-events", groupId = "search-group")
public void handleRoomEvent(RoomEvent event) {
    syncToElasticsearch(event);
}
```

#### Event Sourcing 패턴
```java
// 이벤트 저장소
@Entity
public class RoomEventStore {
    @Id
    private String eventId;
    private Long roomId;
    private String eventType;
    private String payload;
    private Instant timestamp;
}

// 이벤트를 저장하고 발행
@Transactional
public Room createRoom(RoomDto dto) {
    Room room = roomRepository.save(dto.toEntity());
    
    // 이벤트 저장
    RoomEventStore event = new RoomEventStore(room);
    eventStoreRepository.save(event);
    
    // Kafka 발행
    kafkaTemplate.send("room-events", event);
    
    return room;
}
```

#### 장애 처리
```java
@Configuration
public class KafkaConfig {
    @Bean
    public ProducerFactory<String, Object> producerFactory() {
        Map<String, Object> config = new HashMap<>();
        config.put(ProducerConfig.ACKS_CONFIG, "all");  // 모든 복제본 확인
        config.put(ProducerConfig.RETRIES_CONFIG, 3);
        config.put(ProducerConfig.ENABLE_IDEMPOTENCE_CONFIG, true);
        return new DefaultKafkaProducerFactory<>(config);
    }
}
```

#### Dead Letter Queue (DLQ)
```java
@KafkaListener(topics = "room-events")
public void handleRoomEvent(RoomEvent event) {
    try {
        processEvent(event);
    } catch (Exception e) {
        // DLQ로 전송
        kafkaTemplate.send("room-events-dlq", event);
        log.error("Failed to process event: {}", event, e);
    }
}
```
