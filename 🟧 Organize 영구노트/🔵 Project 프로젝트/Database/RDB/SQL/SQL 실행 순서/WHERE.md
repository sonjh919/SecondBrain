## WHERE
+ [[🟡 Area/Database/RDB/SQL/DML/SELECT]]문에 WHERE절을 추가하여 특정 조건에 부합하는 데이터들만 조회할 때 사용한다.
+ 여러 [[🟡 Area/Database/RDB/SQL/문법/연산자]]들을 사용하여 조건을 설정할 수 있다.
```sql
SELECT * FROM singer
  WHERE genre = 'kpop';
```
