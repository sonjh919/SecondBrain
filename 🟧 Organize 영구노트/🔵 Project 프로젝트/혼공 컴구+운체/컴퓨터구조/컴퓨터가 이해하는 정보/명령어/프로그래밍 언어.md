#컴퓨터구조 

### 고급 언어
+ 사람을 위한 언어

> [!example]+ 
> Java, C, Python ...

### 저급 언어
+ 컴퓨터가 직접 이해하고 실행할 수 있는 언어

> [!example]+ 
> [[기계어]], [[어셈블리어]]
### 고급 언어 → 저급 언어
고급 언어가 저급 언어로 변환되어 실행되는 것은 크게 2가지 방식이 있다.

> [!summary]+ 
> + [[컴파일]] 방식
> + [[인터프리터]] 방식

> [!check]+ 
> 하나의 프로그래밍 언어가 반드시 둘 중 하나로만 작동되지는 않는다.
