### 생성자
컨트랙트와 동일한 이름을 가지며, 생략할 수 있는 특별한 [[3. Resource/BlockChain/솔리디티(Solidity)/Beginner/Lesson 1 - 좀비 공장 만들기/함수|함수]]이다.
이 함수는 컨트랙트가 생성될 때 딱 한 번만 실행된다.

```Java
contract Ownable {  
  address public owner;  
  
  function Ownable() public {  
    owner = msg.sender;  
	 }
  }
```
